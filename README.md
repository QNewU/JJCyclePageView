# JJCyclePageView
一个能滚到世界尽头的PageView
# CocoaPods
 pod 'JJCyclePageView', '~> 1.0.7'
# 功能
<br>可以横着滚
<br>也能竖着滚
<br>还能自己滚
<br>可以从左往右自己滚
<br>也能从右往左自己滚
<br>可以从上往下自己滚
<br>也能从下往上自己滚
<br>可以滚一个
<br>也能滚多个
<br>还能不同种类的一起滚
